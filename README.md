# Postiflow

Postiflow is a Next.js 15+ SaaS starter for building social-media scheduling and posting workflows. It ships with the App Router, TypeScript, Tailwind CSS, shadcn/ui components, and a Supabase client wired up for browser and server contexts — ready to be extended with auth, queues, and billing in later phases.

## Setup

1. Install dependencies: `npm install`
2. Copy env template and fill in Supabase credentials: `cp .env.local.example .env.local`
3. Run the dev server: `npm run dev`
4. Open http://localhost:3000
5. Build for production: `npm run build`
